portfolio

Portfolio Website

このポートフォリオサイトです。 AWS EC2のUbuntuサーバで構築しました。NGINXを使用してHTTPリクエストを処理し、 UFWでHTTP/HTTPS（ポート80/443）とSSH（ポート22）を開放しています。 ドメインはお名前.comで管理し、DNS設定でEC2のIPアドレスを正しく指すようにしました。 また、CertbotでLet’s EncryptのSSL証明書を取得し、HTTPS接続を実現しました。

Online Chat Messenger(チーム開発)

ユーザーはチャットルームを作成したり、既存のルームに参加したりして、 他のユーザーとリアルタイムでコミュニケーションを取ることができます。 通信はTCPとUDPプロトコルを使用し、RSA暗号化によりデータの安全なやり取りを実現しています。

Video Processer

ユーザーが動画ファイルをアップロードし、以下の機能 (圧縮、解像度の変更、アスペクト比の変更、音声に変換、指定した時間範囲でGIFの作成)を実行できるプログラムです。 このプログラムでは、FFMPEGライブラリを用いてタスクを実行し、品質と互換性を確保しています。

旅行先みくじ(チーム開発)

トップページのボタンをクリックすると、おすすめの名物・名所に一言添えて旅行先がランダムに表示されます。

Web API(技術課題)

最安値上映スケジュール検索システムは、現在日時と参加者の構成を入力すると、最も安く観覧できる上映日時と料金を返すシステムです。本システムは、技術課題(1週間)を満たすために作成しました。

Markdown to HTML Converter

任意のマークダウンファイルを HTML ファイルに変換するプログラムです。
